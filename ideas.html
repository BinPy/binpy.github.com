Google Summer of Code 2014 Ideas


1. Circuit Design Software

Description : Creating a GUI using Python and Qt where you can directly drag and drop elemets like wires , gates, ICs, etc and connect them to our Python code. This will reduce the amount of programming to be done by the user by a huge margin so that even amature programmers and even the hardware designers can start off easily.
Ultimate goal of this software is to generate Verilog HDL code form the design itself.

Requirements : Experience in Qt and python. Mentors: Sarwar Chahal and Abhinav Gupta


2. Simulation

Description : Create a graphic simulation of the built circuit that compiles the code and displays the features of the circuit in real-time just like Xilinx and ModelSim software.

Requirements : Experience in Qt and python. Mentors: Sudhanshu Mishra and Abhinav Gupta


3. Extending Libraries

Description : The scope of algorithms and gates used in hardware design are numerous.Some basic work has been done and important gates and algorithms have been implemented but we would like to further expand our libraries to various other aspects of digital electronics.

Requirements : Experience in python and hardware design. Mentors: Sarwar Chahal and Sudhanshu Mishra


These ideas are just for reference but we would like developers to come up with their own ideas on various algorithms and circuits that are being used in harware design so that user can start developing on top of these.
